# ⏳ ZenTime: Premium Pomodoro & Focus Timer

ZenTime is a modern, high-performance productivity tool built with **React** and **Web Audio API**. It transforms the classic Pomodoro technique into a sleek, minimalist experience designed to keep you focused and in the flow.

---

## ✨ Key Features

- 🍅 **Pomodoro Presets**: One-click starts for classic 25m Work, 5m Short Break, and 15m Long Break sessions.
- 🪄 **Dynamic Progress Ring**: A beautiful, animated circular SVG that winds down as your time decreases.
- 🔊 **Web Audio Alarms**: Harmonious triple-chime notifications built directly into the browser (no external assets required).
- 🌓 **Glassmorphism UI**: A stunning dark-mode aesthetic with backdrop blurs and subtle glows.
- 📈 **Streak Tracking**: Keep track of your productivity sessions with an integrated Pomodoro counter.
- 📱 **Fully Responsive**: Optimized for desktops, tablets, and smartphones.
- 🔗 **Tab Sync**: Real-time timer updates directly in the browser tab title.

---

## 🛠️ Built With

- **React** (Hooks: `useState`, `useEffect`, `useRef`)
- **Web Audio API** (Synthesized alert tones)
- **Vanilla CSS3** (Custom glassmorphism design system)
- **SVG** (Dynamic circular progress tracking)

## 📦 Getting Started

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/AaqibhafeezKhan/Timer-Countdown-using-React-Hooks.git
   ```

2. **Install dependencies**
   ```bash
   cd Timer-Countdown-using-React-Hooks
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
